# 12 — Code Assistants

Learn to build and work with highly agentic coding assistants — using Claude Code
as the primary tool and extending it with custom agent skills.

## 🎯 Learning Objectives

- Use Claude Code to navigate, edit, and refactor real codebases
- Drive multi-step engineering tasks with an agentic assistant
- Understand tools, permissions, and the agent loop in coding contexts
- Build and package **agent skills** to extend assistant capabilities
- Apply coding assistants to your own projects in this repo

## 🧩 Key Concepts

- **Agentic coding** — plan → edit → run → verify loops
- **Tools & permissions** — how the assistant acts safely on a repo
- **Skills** — reusable, packaged capabilities for an agent
- **Context management** — working effectively within a large codebase
