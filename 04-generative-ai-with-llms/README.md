# 04 — Generative AI with LLMs

A deeper, lifecycle-oriented view of working with large language models — from
pre-training through fine-tuning, evaluation, and deployment.

> 📦 **Course:** [Generative AI with LLMs](https://www.deeplearning.ai/courses/generative-ai-with-llms) · ~20 hrs · Jun 22 – Jun 26

## 🎯 Learning Objectives

- Understand the generative AI project lifecycle
- Distinguish pre-training, fine-tuning, and in-context learning
- Understand instruction tuning and RLHF at a high level
- Know how to scope, evaluate, and deploy an LLM solution
- Reason about cost, latency, and model selection

## 🧩 Key Concepts

- **Pre-training** — learning language from massive corpora
- **Fine-tuning** — adapting a model to a task or domain
- **PEFT / LoRA** — efficient fine-tuning techniques
- **RLHF** — reinforcement learning from human feedback
- **Evaluation** — benchmarks, metrics, and human review
- **Inference parameters** — temperature, top-p, max tokens

## 🔄 The Lifecycle

```
Define use case → Choose model → Adapt (prompt / fine-tune)
        → Evaluate → Deploy → Monitor & iterate
```
