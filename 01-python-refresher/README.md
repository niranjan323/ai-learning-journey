# 01 — Python Refresher

Refresh the Python fundamentals you need for everything that follows: variables, data types,
collections, control flow, and functions.

> 📦 **Course:** [AI Python for Beginners](https://learn.deeplearning.ai/courses/ai-python-for-beginners) · ~30 hrs · self-paced (ongoing)

## 🎯 Learning Objectives

- Be comfortable with Python syntax and the REPL
- Use variables and core data types confidently
- Work with lists, tuples, dictionaries, and sets
- Control program flow with loops and conditionals
- Write and reuse functions
